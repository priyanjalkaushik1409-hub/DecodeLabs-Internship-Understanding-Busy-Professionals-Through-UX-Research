# Understanding Busy Professionals Through UX Research

## Overview
This project explores the daily challenges, behaviors, and needs of busy professionals through UX research. The objective was to identify pain points related to productivity, time management, work-life balance, and digital tool usage.
The research findings were documented and presented in a Canva presentation, highlighting key user insights and potential opportunities for user-centered design solutions.

## Project Goals
- Understand the routines of busy professionals.
- Identify common productivity challenges.
- Analyze user needs and frustrations.
- Generate actionable UX insights.
- Practice UX research and presentation skills.

## Research Process
1. Problem Definition
2. User Research
3. Data Collection
4. Insight Analysis
5. Findings & Recommendations
6. Presentation Creation in Canva

## Key Deliverables
- UX Research Findings
- User Insights
- Pain Points Analysis
- Recommendations
- Canva Presentation
  
## Tools Used
- Canva
- User Analysis
  
## Learning Outcomes
This project helped me develop foundational UX research skills, including user understanding, problem identification, data synthesis, and communicating findings through a professional presentation.
